# 📱 Contact Manager (Glass UI)

A modern, responsive, and interactive Contact Manager built using HTML, CSS, and JavaScript with a premium glassmorphism design.

---

## 🌐 Live Demo

👉 https://kumarsubodh12.github.io/contact-manager/

---

## ✨ Features

* 📇 Add & delete contacts
* 🔍 Real-time search functionality
* 💾 Data stored using Local Storage (no data loss on refresh)
* 🌙 Smooth UI interactions & animations
* 💎 Glassmorphism design (blur + transparency)
* 🎯 Button click animations
* 📱 Fully responsive (mobile + desktop)

---

## 🎨 UI Highlights

* Sky blue & white theme ☁️
* Soft shadows & glass effects ✨
* Smooth hover animations 😎
* Emoji-based interactive design 🎉

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Glass UI + animations)
* JavaScript (DOM + Local Storage)
